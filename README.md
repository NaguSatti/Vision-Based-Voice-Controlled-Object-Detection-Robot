# Vision-Based-Voice-Controlled-Object-Detection-Robot
Developed an intelligent robotic system integrating computer vision and voice recognition for real-time object detection and navigation.
Implemented object detection using OpenCV and voice command processing using Python speech recognition libraries.
The robot can identify objects and respond to voice commands such as forward, backward, left, and right.
Technologies Used:

•Python

•OpenCV

•Raspberry Pi

•Arduino

•Speech Recognition

•Embedded Systems
